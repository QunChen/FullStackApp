# FullStackApp

to run the app: 
  1. open a terminal  cd json-server  
  
  2. json-server --watch db.json
  
  3. open another terminal  cd ../ 
  
  4. gulp watch

to test the app: 

   unit: 
   
1. cd test 

2. karma start karma.conf.js


   e2e: 
   
1. cd test

2. protractor protractor.conf.js (make sure the app is running first)
